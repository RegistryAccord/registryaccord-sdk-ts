# Feature Development Prompt

**Context**: Implement a new feature for the RegistryAccord SDK.

**Instructions**:
1. Read the specific service requirement file in `docs/requirements/`.
2. Check `.ai/knowledge/tech-stack.md` for architectural patterns.
3. Verify against `docs/scenarios/` if applicable.
4. Create a plan in `memory-bank/activeContext.md`.
5. Write failing tests in `tests/`.
6. Implement the feature in `src/`.
7. Verify all tests pass.
8. Update the Version History table in the requirement file after changes.
